# PROJECT CONTEXT
**Game Type:** Mobile Rogue-lite with meta-progression.
**Orientation:** Landscape.
**Core Gameplay Loop:** Choose 3 characters -> Start Run -> Select Level -> Fight -> Next Level.
**Key Mechanics:**
- Boss encounters at the end of each zone.
- Upon death, the player gains resources to buy meta-upgrades.
- Spells can be unlocked via achievements or purchases.
- Each character has 4 base spells. Additional spells drop during runs.
