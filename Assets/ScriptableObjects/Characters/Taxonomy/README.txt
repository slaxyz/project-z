Character Taxonomy (single source of truth)

Folders:
- Rarities: rarity_0..rarity_4
- Types: type_1..type_6
- Roles: role_1..role_6
- Classes: class_*
- Passives: passive_*

Workflow:
1) Import/update data with:
   Tools > Project Z > Import Characters From Excel
2) Link icons/backgrounds in taxonomy assets (if needed).
3) Run check:
   Tools > Project Z > Audit Character Data

Mappings:
- Rarity: 0 Default, 1 R, 2 SR, 3 SSR, 4 LR
- Type: 1 Fire, 2 Nature, 3 Poison, 4 Water, 5 Normal, 6 Mystic
- Role: 1 Warrior, 2 Tank, 3 Rogue, 4 Healer, 5 Specialist, 6 Gunner
